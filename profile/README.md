## 香港服务器的那些"行话"，先搞清楚

选香港服务器绕不开几个线路名词，不懂的话很容易买错：

**CMI（China Mobile International）**：中国移动国际线路，也叫 CMIN2，专为移动用户直连优化，延迟表现出色，丽萨主机香港三网产品的移动去程就走这条线。

**CN2 / CU2**：中国电信CN2精品网和中国联通CU2优化线路，分别对电信和联通用户优化。三网直连方案同时涵盖这三条线路，无论你用哪家运营商都不绕路。

**HGC / iCable**：香港本地运营商（香港宽频HGC、有线宽屏iCable），提供原生住宅IP，纯净度高，可解锁所有港区本土服务，包括TVB直播、Cityline票务、Netflix香港区等。
